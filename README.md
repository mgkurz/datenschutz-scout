# datenschutz-scout
Interaktive Lernübung zum Thema Datenschutz und Datenmissbrauch-Vermeidung für die Mittelstufe (Klasse 7–10). Single-File HTML5, keine externen Abhängigkeiten. Optimiert für die Einbettung per iframe in Moodle.
Datenschutz-Scout – Interaktive Lernübung
Schülerinnen und Schüler lernen spielerisch, persönliche Daten im Netz zu schützen. Szenariobasiert, mit Punktesystem und gestuften Hilfen.
Zielgruppe: Mittelstufe, Klasse 7–10 (Realschule/Gymnasium)
Bearbeitungszeit: ca. 20–30 Minuten
Vorwissen: Keins erforderlich
Inhalte

Datenkategorien unterscheiden (harmlos, sensibel, hochsensibel)
Daten-Check: 4 Fragen vor jeder Dateneingabe
5 Alltagsszenarien (Apps, Gewinnspiele, Social Media, KI-Chatbots, Gaming)
Hilfe-Netzwerk: An wen kann ich mich wenden?
Sicherungsmodul mit Ergebnisübersicht, Druck- und Markdown-Export

Einsatz
Direkt im Browser öffnen oder per iframe in Moodle einbetten:
html<iframe src="https://DEIN-USERNAME.github.io/datenschutz-scout/" width="100%" height="800" frameborder="0"></iframe>
Technik
Single-File HTML5 (HTML + CSS + JS). Keine externen Abhängigkeiten. Offline-fähig. Touch-optimiert für Tablets.
